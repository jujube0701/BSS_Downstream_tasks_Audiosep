# AudioSep 下游任務專案

這個專案使用 AudioSep 模型進行音訊盲源分離的下游任務，包括環境音分類、樂器分類和說話人識別。

## 🎯 任務說明

### 1. ESC-50 環境音分類
- **資料集**: ESC-50 環境音分類資料集
- **任務**: 使用 AudioSep 提取的特徵進行環境音分類
- **模型**: 單層和三層全連接分類器
- **特徵**: 使用 AudioSep 模型提取的音訊特徵

### 2. NSynth 樂器分類
- **資料集**: NSynth 合成樂器資料集
- **任務**: 樂器家族分類
- **資料分割**: 使用 valid 和 test 集
- **模型**: 單層和三層全連接分類器

### 3. LibriSpeech 說話人識別
- **資料集**: LibriSpeech 語音資料集
- **任務**: 說話人識別
- **特徵**: AudioSep 提取的音訊特徵
- **模型**: 單層和三層全連接分類器
